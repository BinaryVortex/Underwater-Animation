## Overview

This small project demonstrates a looping, animated underwater scene implemented with plain HTML, CSS and JavaScript. It's lightweight, easy to read, and a good starting point for learning DOM-based animations and CSS transitions.

## Features

- Animated background and elements to simulate underwater movement
- Smooth, performant animations using CSS and requestAnimationFrame
- Easy to modify and extend (add fish, bubbles, parallax layers)

## Files

- `index.html` - The demo page. Open this file in your browser to view the animation.
- `script.js` - JavaScript that drives the animations and interactions.
- `Screenshot 2024-07-09 070244.png` - Demo screenshot used in this README.

## How to run

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Underwater-Animation.git

2. Open `index.html` in your browser (double-click or use `file://`), or serve it with a simple HTTP server:

   - Python 3: `python -m http.server 8000` then open `http://localhost:8000` in your browser.
   - Node (http-server): `npx http-server .` then open the printed URL.

No build step or dependencies are required.

## Customize

- Tweak CSS variables and keyframes in the stylesheet to change colors and speeds.
- Edit `script.js` to add new entities (fish, plants) or change movement behavior.
